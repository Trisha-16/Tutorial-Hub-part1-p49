CSS stands for Cascading Style Sheets.

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

HTML stands for Hyper Text Markup Language.

  • HTMLs are generally used to describe a web page's appearance(CSS) or behaviour(Javascript).
  • HTML uses "markup" to annotate text, images other content for display in a web browser.
  • An HTML element is set off from other text in a document by "tags", which consist of the element name surrounded by "<" and ">".

JS stands for JavaScript.

  • JavaScript is an interpreted compiled programming language.
  • JavaScript is a well-known scripting language for Web pages.
  • JavaScript is an easy to learn and also powerful scripting language, widely used for controlling web page behavior.
